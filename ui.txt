HOW IT LOOKS:
- tabletop
- each client starts in a certain state:
    - 2 possible states:
        - active
        - spectating
    - if active --> client sees private view
    - if spectating --> client sees full state view
- for each active client:
    - chat feature, shows up as black box next to client icon
        - this is where LLM thoughts show up
        - client can also type in their own chat box, whatever they put there is public view
        - spectating clients have no chat box and are ejected from public view
    - cards view:
        - appear face-down
        - client can click the cards, this plays an animation that lifts the cards face-up
        - for card selection:
            - cards automatically go face-up for client; depending on state, client picks 1-2 cards.
    - actions view:
        - only the possible actions (including illegal actions but not including impossible actions such as Assassinate when num_coins < 3, etc)
        - shows each action card
        - shows up as empty unless it's the client's turn
        - shows response actions if there is space for client to respond to an action
- for all clients:
    - discard pile:
        - appears next to deck
        - client can click on discard pile and see all the cards in there
    - deck:
        - stack of card jpgs, cannot interact with it
    - coins:
        - single coin icon with number of coins next to it, appears next to client cards
    - active players:
        - show player icons with their names next to them
    - when a player is out, they become a spectator; active players can no longer see them on the tabletop

- scorebaord box type thing to keep track of player cards and eliminated players
